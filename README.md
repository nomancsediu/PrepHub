## Overview

PrepHub is a full-stack bilingual learning platform. Content is organized into subjects, topics, and lessons — available in both English and Bangla. An AI assistant (powered by Groq LLaMA) is embedded on every lesson page to answer questions in context.

---

## Core Features

<div align="center">

| Feature | Description |
|---|---|
| Bilingual Content | Every lesson supports English and Bangla with a one-click toggle |
| AI Assistant | Context-aware chat powered by Groq LLaMA on every lesson |
| AI Translation | Write in Bangla — AI auto-translates to English in the admin panel |
| Lesson Likes | Users can like lessons; count persists in the database |
| Visit Counter | Live site visit count displayed in the navbar |
| Admin Panel | Full CRUD for subjects, topics, and lessons with rich text editor |
| Publish Control | Mark subjects as draft or published to control visibility |

</div>
